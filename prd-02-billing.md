## 3. 积分消耗

### 3.1 积分分类

| 类型               | 来源                 | 有效期             |
| ------------------ | -------------------- | ------------------ |
| **免费积分** | 新用户注册赠送       | 永久有效           |
| **订阅积分** | 订阅套餐每月周期补发 | 1 个月（到期清零） |
| **充值积分** | 用户单独购买         | 1 年               |

> **消耗顺序**：免费积分 → 订阅积分 → 充值积分

### 3.2 消耗规则

| 操作                  | 消耗积分  | 备注                       |
| --------------------- | --------- | -------------------------- |
| AI 视频生成（15s）    | ⚠️ 待定 | 含去背 + 提示词增强 + 渲染 |
| AI 视频生成（60s）    | ⚠️ 待定 | 图片 + 文字生成            |
| AI 局部重绘（单片段） | ⚠️ 待定 | 视频 + 文字生成视频        |
| 字幕生成              | ⚠️ 待定 | 是否消耗积分待定           |
| 模板克隆              | ⚠️ 待定 | —                         |

> 参考换算：免费版 120 积分 ≈ 1 条视频，Pro 1300 积分 ≈ 5 条视频，推算单次约消耗 120 积分/条

## 6. 续费 / 到期处理

### 6.1 套餐升降级

| 场景                        | 规则                                                         |
| --------------------------- | ------------------------------------------------------------ |
| 免费 → Pro/Plus/Max        | 随时升级，即时生效                                           |
| Pro → Plus/Max（包年中途） | ⚠️ 待定：方案 A 不支持 / 方案 B 剩余有效期按比例退款后升级 |
| 降级                        | 不支持                                                       |

### 6.2 到期行为

- 订阅到期后，订阅积分**立即清零**，不可结转
- 免费积分与充值积分**继续有效**
- 降级为免费版，历史生成记录保留（只读）
- 续订后下个周期月初重新补发订阅积分

### 6.3 退款政策

> ⚠️ 待定，参考行业通行：购买后 7 日内且积分未消耗超过 10% 可申请全额退款
